The Cartogram Maker

The easy way to generate cartograms.

Instructions for use:

The program has a very simple workflow that is described below.

  • Upload the map file that you want to transform (Note: must be either in an SVG or GeoJSON file format).
  • Upload the file containing the corresponding data points for that map file.
  • Select continue at the bottom of this page.
  • This will display a preview of your uploaded map file. If you like the preview you can then select continue or back if not. The system will then compute the cartogram using the Gastner-Newman algorithm.
  • The system similarly to the previous step will preview the cartogram it has created.
  • If you like what you see, click download to download the file
  • Select return to return back to this page.
