Instructions

1. File Upload

First step is to upload your SVG map file and CSV data file. A window will open where you can select which files to upload to the server. An example of the map file and data file that the system expects can be downloaded below.


The CSV file has a very simple format. Each line has two elements that represent the name of the region and its desired statistic. The SVG file has a more detailed requirement. Each region in the map file must have a name associated with it, if this is not the case the system will reject it. If a region in the map file doesn't have a data value associated with it, the region will remain the same in the resulting cartogram. For the d attribute, the system only supports C, L, M or Z characters, it will reject other characters. If a file does contain a C character, its recommended using other software to convert the curves to line segments as it takes the system a substantial length of time to do this.

Press Next to Continue!!
