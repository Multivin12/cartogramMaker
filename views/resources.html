Resources

The resource that was used to implement the cartogram algorithm can be found here.

It provides a C code implementation of Gastner and Newman's diffusion based algorithm and was written by one of the algorithm's creators, Mark Newman. To find out more about this algorithm, their paper can be found here.

Overview of the Algorithm

The algorithm works by diving a map into sections by projecting a quadrilateral grid onto it.



It calculates a density value for each grid square and then the algorithm diffuses the grid until there is uniform density everywhere.



This grid is then applied onto the map which intern produces the resulting cartogram.
